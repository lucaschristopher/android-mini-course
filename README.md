### Android Mini Course

Repository dedicated to the practical of Android programming. It is a simple application for listing books published in the New York Times API, using the Kotlin language, MVVM as an architectural pattern and Retrofit for our request.

_Content by_: [Douglas Motta channel](https://www.youtube.com/playlist?list=PLPs3nlHFeKTrD1az_BrbIuXZi3EOfMqNv);
